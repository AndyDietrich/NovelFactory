## Configuration

### **API Providers**

#### OpenRouter (Recommended)
- **Website**: [openrouter.ai](https://openrouter.ai)
- **Models**: Claude Sonnet 4, Claude Opus 4.1, GPT-5, GPT-4o
- **Benefits**: Access to multiple AI providers, competitive pricing

#### OpenAI
- **Website**: [platform.openai.com](https://platform.openai.com)
- **Models**: GPT-5, GPT-4o, GPT-4 Turbo, GPT-4o Mini
- **Benefits**: Direct access to latest OpenAI models

### **Recommended Settings**
- **Model**: Claude Sonnet 4 (best quality & affordable). GPT-5 Mini (most economical) or Opus 4.1 for peak performance and price
- **Temperature**: 0.7 (balanced creativity)
- **Max Tokens**: 50,000 (sufficient for most chapters)
- **Feedback Loops**: 1-2 for quality improvement

## Usage Guide

### **Step 1: Book Setup**
- Choose your genre (Fantasy, Romance, Mystery, etc.)
- Select target audience (Children, YA, Adult, etc.)
- Enter a compelling premise (2-3 sentences)
- Describe your desired writing style
- Set chapter count and target word count per chapter

### **Step 2: Story Structure**
- AI generates complete narrative framework
- Includes character development and plot points
- Three-act structure with proper pacing
- Use feedback loops to refine the structure

### **Step 3: Chapter Planning**
- Detailed chapter-by-chapter breakdown
- Scene structure and plot beats for each chapter
- Automatically generates book title and blurb
- Maintains story consistency throughout

### **Step 4: Writing**
- Generate chapters individually or in batches
- Use Expand Mode for distraction-free editing
- Apply feedback loops to improve chapter quality
- Real-time word count and progress tracking

### **Step 5: Export**
- Download in multiple formats (TXT, HTML, MD)
- Copy to clipboard for easy sharing
